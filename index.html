📋 Google Sheets 연동 설정 방법

  1. Google Drive에서 새 스프레드시트를 만드세요.
  2. 상단 메뉴 확장 프로그램 → Apps Script 클릭
  3. 에디터의 기존 코드를 모두 지우고, 제공된 Code.gs 내용을 전체 붙여넣기 후 저장(💾)
  4. 상단 배포 → 새 배포 클릭
  5. 유형: 웹 앱 선택
  6. 다음 사용자로 실행: 나 / 액세스 권한: 모든 사용자 로 설정
  7. 배포 → Google 계정 권한 허용 → 표시된 웹앱 URL 복사
  8. 이 페이지 상단 입력창에 붙여넣기 → 연결하기 클릭
💡 코드 수정 후 재배포할 때: "새 배포"가 아니라 배포 관리 → ✏️ 편집 → 버전 "새 버전" 으로 업데이트해야 URL이 유지됩니다.
⚠️ 저장 실패 시 체크리스트
1. 액세스 권한이 "모든 사용자" 로 설정되어 있는지 확인
2. 배포 후 새 버전이 아니라 기존 버전 URL을 사용 중인지 확인
3. URL이 /exec 로 끝나는지 확인 (/dev 는 로그인 필요)
